An app for browsing currently popular movies and favorites list using the TMDB API.

**Features**
1. Popular Movies: Displays a grid of movies fetched from the TMDB API.
2. Search: Allows searching for movies by title.
3. Favorites System: Uses React Context and local storage for persisting favorited movies.

**Tech Stack**
- Frontend: React (Hooks)
- Build Tool: Vite
- Routing: react-router-dom
- State: React Context API
- Data: The Movie Database (TMDB) API


Node.js and npm (or yarn/pnpm) are required.

**Start the development server:

npm run dev
**
